# 📢 Git 컨벤션

## ✨ Commit Message 형식

| 규칙                     | 설명                                       |
|--------------------------|--------------------------------------------|
| feat        | 새로운 기능 추가한 경우 |
| fix       | 코드 수정     |
| design               | CSS 사용자 UI 디자인 변경            |
| style            | 코드포맷 변경, 세미콜론 누락, 코드 수정 없는 경우          |
| refactor               | 코드 리팩토링      |
| comment               | 주석 추가 및 변경       |
| rename               | 파일,폴더명 변경        |
| remove               | 파일만 삭제        |
| add               | 새로운 파일만 추가        |

## 🎨 Branch Naming 형식

- #feat(숫자)/**
- #fix(숫자)/**
